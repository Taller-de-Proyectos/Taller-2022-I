Descripcion del proyecto

Virtual Bot - La Carrera Semanal

Este proyecto aprovecha las tecnologías de IOT, NPL, Reconocimiento Facial, Realidad Virtual, Página Web y Digital Twins que nos proporcionan las nubes de Google Cloud, Azure Cloud, y la herramienta Unity y WebGL, donde todo será presentado en una página web y en el que tendrá una interacción directa con el alumno del curso de taller de proyectos de la facultad de la FIA. Este proyecto busca medir las capacidades físicas de los estudiantes de la USMP. Donde un avatar realizado con realidad virtual conectado al chatBot hablará con el usuario, este le realizará una serie de preguntas relacionadas con su edad, peso, talla, actividad. La tecnología de IOT trabajará con los sensores del celular, donde capturará la distancia, tiempo y ritmo del recorrido realizado por el usuario, dicha información viajará a una base de datos y será mostrada en la página web a modo de reporte. Para poder visualizar el reporte de la carrera, se empleará la tecnología de reconocimiento facial para brindar el acceso a dicha interfaz.

Arquitectura

Descripcion

La arquitectura global nivel 2 del trabajo muestra todos los componentes utilizados durante la ejecución del proyecto. Se observa el uso de componentes para las tecnologías de IOT, NPL, Rec. Facial, Realidad Virtual, Página Web y Digital Twins Donde se trabajó en 2 nubes, tales como la de Google Cloud y Azure Cloud, y la gestión del proyecto en Azure DevOps, además la aplicación de CI/CD con DevOps y Google Platform. Finalmente se puede observar la interacción entre todos los componentes y la integración de estos entre las 2 nubes, todo esto mostrado en la página web del proyecto final.
